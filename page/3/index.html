42.vue3项目总结

发表于 2021-10-20 更新于 2021-10-25 分类于 项目实战

1.配置Header组件和Tab组件

Header组件包括icon和标题，右边包括用户中心图标，点击跳转到用户中心

Tab组件包括Recommend组件、Singer组件和TopList组件，打开主页默认重定向到Recommend

将Header和Tab组件挂载到App.vue上

2.推荐页面

1.使用封装好的Scroll组件

2.使用封装好的Slider组件,将sliders数据作为props传入

3.使用Albums组件，将albums数据传入，并使用自定义指令v-loding，当点击歌单路由进行跳转 (点击时会向Recommend组件发出一个事件，并传入点击时的歌单数据，recommend接收到事件跳转到相应路由并将歌单Id作为url参数，然后将数据传入给router-view，router-view封装了歌曲详情页.)

4.router-view对应了AlbumDetail组件，所有Detail页面(AlbumDetail，SingerDetail，TopDetail)都封装了DetailList组件和createDetailComponent逻辑。

3.歌手页面

歌手页面包括三部分: 顶部固定栏、歌手列表、右边字母表

顶部固定栏的吸顶效果

1.计算出每个歌手组的高度并放入一个数组

2.获取当前滚动高度并watch，如果高度在某个歌手组之间则设定为当前歌手组的顶部栏

3.计算出顶部栏距离顶部高度distance，如果顶部栏在0—顶部栏高度之间，计算出偏移量diff,并设置translate3d(0,diff,0)把上一个顶部栏顶出去

字母表的切换

1.将每个数据的title作为一个数组返回

2.利用touchstart计算出当前滚动的索引，并保存索引和pageY偏移量，用当前索引计算出对应元素，利用scroll实例的scrollToElment滚动到相应位置

3.利用touchmove和touchstart的索引以及偏移量，计算出滚动时的索引位置，用当前索引计算出对应元素，利用scroll实例的scrollToElment滚动到相应位置

TypeScript-基础类型

发表于 2021-10-10 更新于 2021-10-27 分类于 TypeScript

简介

  • TypeScript 是添加了类型系统的 JavaScript，适用于任何规模的项目。
  • TypeScript 是一门静态类型、弱类型的语言。
  • TypeScript 是完全兼容 JavaScript 的，它不会修改 JavaScript 运行时的特性。
  • TypeScript 可以编译为 JavaScript，然后运行在浏览器、Node.js 等任何能运行 JavaScript 的环境中。
  • TypeScript 拥有很多编译选项，类型检查的严格程度由你决定。
  • TypeScript 可以和 JavaScript 共存，这意味着 JavaScript 项目能够渐进式的迁移到 TypeScript。
  • TypeScript 增强了编辑器（IDE）的功能，提供了代码补全、接口提示、跳转到定义、代码重构等能力。
  • TypeScript 拥有活跃的社区，大多数常用的第三方库都提供了类型声明。
  • TypeScript 与标准同步发展，符合最新的 ECMAScript 标准（stage 3）。
